**Tweet 5:**

**Tweet 6:**
The wildest feature: AI-powered website conversion.

Run `terminaltui convert` in any web project. It drops reference docs that let Claude (or any AI) read your site and produce a working TUI version.

I've tested it on dozens of sites. It works shockingly well.

**Tweet 7:**
This isn't just a static site generator. It's a real app framework:

- Reactive state (get/set/subscribe/batch)
- 10 input components (text, select, checkbox, radio, search)
- Forms with validation and submission
- Parameterized routing with middleware
- WebSocket and SSE connections
